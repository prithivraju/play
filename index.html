Getting Started

To run this application locally:

  1. Clone the repository: git clone https://github.com/prithivraju/play.git
  2. Install dependencies: npm install
  3. Add your Claude API key to environment variables
  4. Start the development server: npm start
  5. Upload a PDF and choose your learning mode
